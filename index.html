☁️ Firebase Cloud Sync
📋 Cara Setup (4 langkah):
1. Buka console.firebase.google.com → buat project
2. Pilih Realtime Database → Create Database → pilih region → Test mode
3. Salin URL database (contoh: https://xxx-rtdb.asia-southeast1.firebasedatabase.app)
4. Paste di bawah → klik Aktifkan
⚠️ Wajib: Di Firebase Console → Realtime Database → tab Rules → ubah jadi:
{ "rules": { ".read": true, ".write": true } }
Lalu klik Publish. Tanpa ini akan selalu gagal.
